keybindings, maestro

global

  • m - toggle menu (when search bar is not active)
  • j - next menu item (when menu is active)
  • k - previous menu item (when menu is active)

home page

  • / - toggle search bar
  • j - next search result
  • k - previous search result
  • o - open video (when search result is active)
  • c - toggle cart status for video (when search result is active)
